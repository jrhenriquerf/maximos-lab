# Checklist de publicação

## GitHub

- [ ] Repositório criado ou atualizado.
- [ ] Branch correta configurada.
- [ ] Arquivos finais publicados.
- [ ] `index.html` presente na raiz/pasta publicada.
- [ ] GitHub Pages ativo.
- [ ] Arquivo `CNAME` correto, se houver domínio customizado.

## Cloudflare

- [ ] Domínio adicionado.
- [ ] Nameservers ativos.
- [ ] SSL/TLS em Full.
- [ ] Always Use HTTPS ativo.
- [ ] Brotli ativo.
- [ ] HTTP/3 ativo quando disponível.

## DNS

- [ ] Apex configurado, se usado.
- [ ] `www` configurado, se usado.
- [ ] Subdomínios configurados, se usados.
- [ ] Registros GitHub Pages corretos.
- [ ] DNS Check do GitHub Pages aprovado.

## SEO

- [ ] Title configurado.
- [ ] Meta description configurada.
- [ ] Canonical configurado.
- [ ] Open Graph configurado.
- [ ] Favicon configurado.
- [ ] Robots.txt publicado.
- [ ] Sitemap.xml publicado quando necessário.

## Performance

- [ ] Imagens otimizadas.
- [ ] Lazy loading aplicado.
- [ ] Scripts externos revisados.
- [ ] Lighthouse executado.

## Testes

- [ ] Desktop conferido.
- [ ] Mobile conferido.
- [ ] Links internos testados.
- [ ] Links externos testados.
- [ ] WhatsApp testado.
- [ ] Formulários testados, se houver.

## Entrega

- [ ] Site aprovado.
- [ ] Domínio final informado.
- [ ] HTTPS ativo.
- [ ] Cliente recebeu links finais.
