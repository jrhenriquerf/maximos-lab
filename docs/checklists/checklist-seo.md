# Checklist de SEO

- [ ] Title único por página.
- [ ] Meta description clara por página.
- [ ] Canonical com URL final HTTPS.
- [ ] Open Graph completo.
- [ ] Twitter Card configurado.
- [ ] Favicon publicado.
- [ ] Theme color configurado.
- [ ] Imagens relevantes com alt.
- [ ] Sitemap.xml criado para site multipágina.
- [ ] Robots.txt criado.
- [ ] Search Console configurado quando contratado.
- [ ] Sitemap enviado no Search Console quando aplicável.
- [ ] Preview de compartilhamento testado.
- [ ] Lighthouse SEO acima de 95 ou justificativa registrada.
