# Checklist de entrega ao cliente

- [ ] Site aprovado pelo cliente.
- [ ] Domínio ativo.
- [ ] HTTPS ativo.
- [ ] Pagamento final recebido, se aplicável.
- [ ] Recibo emitido.
- [ ] Link final enviado.
- [ ] Links de páginas internas enviados quando necessário.
- [ ] Prazo de garantia informado.
- [ ] O que está incluso após a entrega foi explicado.
- [ ] O que não está incluso após a entrega foi explicado.
- [ ] Acessos entregues quando aplicável.
- [ ] Analytics/Search Console/GTM/Clarity compartilhados quando contratados.
- [ ] Projeto registrado no Notion/CRM.
- [ ] Projeto arquivado.
- [ ] Follow-up ou manutenção agendada, se aplicável.
