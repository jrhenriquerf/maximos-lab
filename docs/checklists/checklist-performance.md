# Checklist de performance

- [ ] Imagens convertidas para WebP quando possível.
- [ ] Imagens redimensionadas para o uso real.
- [ ] `loading="lazy"` em imagens fora da primeira dobra.
- [ ] CSS limpo e sem excesso de regras não usadas.
- [ ] JavaScript mínimo e sem bibliotecas desnecessárias.
- [ ] Scripts de terceiros revisados.
- [ ] Lighthouse Mobile executado.
- [ ] Lighthouse Desktop executado quando necessário.
- [ ] Performance acima de 90 ou justificativa registrada.
- [ ] Accessibility acima de 95.
- [ ] SEO acima de 95.
- [ ] Site testado no mobile.
- [ ] Compressão/Brotli ativa na Cloudflare quando aplicável.
- [ ] Cache avaliado quando aplicável.
