# AGENTS.md

Padrões de trabalho para o projeto Maximos LAB.

## Regras gerais

- Não usar frameworks sem autorização.
- Não inserir senhas, tokens ou API keys no repositório.
- Priorizar performance em todas as alterações.
- Priorizar responsividade para desktop, tablet e mobile.
- Priorizar SEO básico em páginas públicas.
- Manter o código simples, claro e fácil de editar.
- Usar WebP para imagens sempre que possível.
- Usar SVG para ícones sempre que possível.
- Não remover metatags de Open Graph sem confirmação.
- Não alterar links principais sem confirmação.

## Cuidados antes de publicar

- Conferir links de WhatsApp, redes sociais e projetos do portfólio.
- Conferir title, meta description, canonical e Open Graph.
- Conferir imagens, favicon e responsividade.
- Rodar uma revisão visual em mobile e desktop.
- Evitar dependências externas desnecessárias.
