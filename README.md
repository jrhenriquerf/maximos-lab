# Maximos Lab

Portfólio estático para GitHub Pages com página inicial, seções de sobre, serviços, portfólio e formulário de contato via `mailto`.

## Publicação

Publique o conteúdo desta pasta pelo GitHub Pages. Para usar um domínio próprio com Cloudflare, aponte o DNS para o GitHub Pages e configure o domínio customizado nas configurações do repositório.
